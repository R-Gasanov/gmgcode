Now there are a variety of different and unique methods that are for directories specifically

Lets look at all of them finally since we have used most of them, but there are some extra ones that we haven't used as well

Dictionary Methods

clear() Removes all the elements from the dictionary

copy() Returns a copy of the dictionary

fromkeys() Returns a dictionary with the specified keys and values

get() Returns the value of the specified keys

items() Returns a list containing a tuple for each key value pairs

keys() Returns a list containing the dictionary's keys

pop() Removes the elemnt with the specified key

popitem() Removes the last inserted key-valye pairs

setdefault() Returns the value of the specified key. If the key does not exist: insert the key, with the
specified value

update() Updates the dictionary with the specified key-value pairs

values() Returns a list of all values in the dictionary
